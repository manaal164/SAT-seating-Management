# Student Registration & Management System

A web application built with **React**, **Tailwind CSS**, **Node.js**, and **MySQL**.  
Students can register, download a PDF after registration, and withdraw if needed. Admins can view all students, remove students, and manage the system securely via a password-protected portal.

## Features
- Student registration and PDF generation  
- Student withdrawal  
- Admin portal with secure login  
- View and remove students  

## Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  

## Installation
1. Clone the repo: `git clone <your-repo-url>`  
2. Install dependencies: `npm install` (both frontend & backend)  
3. Set up MySQL database and update backend config  
4. Start the project: `npm start`
